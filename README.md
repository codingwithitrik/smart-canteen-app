# Smart Canteen Order & Queue System

A Python-based system that allows students to place food orders in advance
and helps the canteen manage order queues efficiently.

## Features
- Advance ordering
- Queue-based processing
- Reduces waiting time
- Works on Linux & Windows

## Technologies
- Python
- File Handling
- CLI Interface
